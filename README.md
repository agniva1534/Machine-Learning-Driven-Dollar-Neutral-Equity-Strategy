# Machine Learning Driven Dollar Neutral Equity Strategy With Volatility Targeting and Monte Carlo Significance Testing

**Overview**

This project develops and rigorously validates a cross sectional dollar neutral long short equity strategy using machine learning. The objective is to evaluate whether nonlinear models can improve stock ranking accuracy and generate economically meaningful cross sectional alpha under realistic backtesting constraints.

Rather than predicting exact stock returns, the framework focuses on relative ordering. Stocks are ranked by predicted excess return each month, and the strategy takes long positions in the top decile and short positions in the bottom decile. Performance is therefore driven by ranking skill, not point forecast precision.

**Methodology**

The strategy is implemented using a strict rolling out of sample framework. Training, validation, and test windows are fixed and applied chronologically to eliminate look ahead bias. Every return in the backtest is generated by a model that had not seen that month’s data during training. This structure reduces overfitting risk and ensures replicability across time.

Portfolio construction enforces equal weighting within the long and short legs and maintains dollar neutrality at the portfolio level. This isolates cross sectional signal and removes systematic market exposure.

To address time varying volatility, portfolio level volatility targeting is applied using a lagged rolling volatility estimate. Returns are scaled to a constant annualized volatility target, and leverage constraints are imposed to prevent excessive risk scaling. This improves comparability across market regimes.

**Signal Evaluation**

Signal quality is evaluated using multiple cross sectional diagnostics. Decile monotonicity testing examines whether average returns increase from the lowest to highest prediction groups. Information Coefficient analysis measures rank correlation between model scores and subsequent returns. Rolling Sharpe stability and drawdown analysis assess temporal robustness. Low out of sample R squared is interpreted in the context of asset pricing, where ranking consistency is more relevant than point forecast accuracy.

**Robustness Framework**

A Monte Carlo simulation framework is implemented to assess statistical credibility. Randomized return paths are generated to construct an empirical null distribution, allowing evaluation of whether observed performance characteristics could plausibly arise by chance. This provides a robustness layer beyond traditional backtesting.
